# PDF 图纸审批托盘助手用户指南

适用对象：主管、工艺、设计师、管理员。

托盘助手是 Windows 本机提醒工具。审批、提交、签名、管理仍在浏览器里的 `PDF 图纸审批` Web 系统完成。

## 安装

由管理员提供安装包后，在本机双击安装。

安装完成后，Windows 通知区域会出现 `PDF 图纸审批托盘助手` 图标。首次启动会打开登录设置窗口。

如果托盘图标没有显示，先点击任务栏右下角的隐藏图标区域查看。

## 首次登录

登录窗口需要填写：

- 审批系统地址：例如 `http://192.168.1.20:8080`。
- 账号：你的审批系统账号。
- 密码：你的审批系统密码。

登录成功后，设置窗口会自动隐藏，托盘助手在后台运行。

## 日常使用

点击托盘图标可以打开菜单。

主管和工艺常用：

- `打开待审核`：打开浏览器进入待审核图纸列表。
- `打开审批工作台`：打开审批系统首页。
- `立即刷新`：立刻检查是否有新的待审核图纸。
- `托盘设置`：重新打开登录设置窗口。
- `退出当前账号`：清除本机登录信息。
- `退出托盘助手`：完全关闭托盘助手。

设计师常用：

- `提交图纸`：打开浏览器进入图纸提交页面。
- `我的签名`：打开浏览器进入签名维护页面。
- `打开审批工作台`：打开审批系统首页。

管理员菜单会额外显示系统管理入口和运维动作。

## 通知规则

主管和工艺登录后，托盘助手会定时检查待审核图纸。

提醒规则：

- 正常在线时约每 30 秒检查一次。
- 服务离线或网络异常时约每 60 秒重试一次。
- 同一张审批图纸只提醒一次。
- 托盘助手重启后，已提醒过的图纸不会重复弹出。
- 一张新任务会打开该图纸详情，多张新任务会打开待办列表。

设计师不会收到审核提醒。

## 重新登录

需要更换账号、服务器地址或密码时：

1. 点击托盘图标。
2. 点击 `退出当前账号`。
3. 点击 `登录设置`。
4. 重新填写服务器地址、账号和密码。

如果密码已修改，旧登录态会在下一次轮询时失效，托盘状态会变为 `登录已过期`。

## 清除已提醒记录

当前版本在点击 `退出当前账号` 时会同时清除已提醒 ID。

清除后重新登录，如果对应图纸仍在待审核列表中，可能会再次弹出提醒。

## 开机自启动

V5 已接入 Tauri autostart 插件能力，但当前第一版托盘窗口尚未提供开关控件。

管理员可以在后续版本中统一开放“开机启动”设置，或通过 Windows 启动项临时配置。

## 卸载

通过 Windows 设置中的“应用和功能”卸载 `PDF 图纸审批托盘助手`。

卸载后不会删除服务端审批数据，也不会影响坚果云图纸目录。

## 常见问题

### 看不到通知

处理顺序：

1. 确认托盘状态是 `在线`。
2. 确认使用的是主管或工艺账号。
3. 打开 Windows 通知设置，确认允许托盘助手发送通知。
4. 点击托盘菜单里的 `立即刷新`。
5. 打开 Web 系统确认确实有待审核图纸。

### 点击通知没有打开页面

先从托盘菜单点击 `打开审批工作台`，确认浏览器能打开审批系统。

如果浏览器能打开但通知不能打开，记录操作时间并交给管理员查看服务日志和托盘版本。

### 服务地址不知道填什么

向管理员确认。常见格式是：

```text
http://服务器局域网IP:8080
```

例如：

```text
http://192.168.1.20:8080
```
