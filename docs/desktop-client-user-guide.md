# Electron 客户端使用说明

## 首次启动

1. 安装 `PDF图纸审批客户端-安装包-0.9.1.exe`。
2. 从桌面或开始菜单启动 `PDF 图纸审批` 客户端。
3. 在“连接审批服务器”页面填写服务器地址，例如 `http://192.168.1.20:8080`。
4. 点击“连接并保存”。
5. 进入登录页后输入账号和密码。

如果管理员发的是便携版文件夹，请打开整个 `PDF图纸审批客户端` 文件夹，并双击 `PDF图纸审批客户端.exe`。不要把 exe 单独复制到桌面运行。

## 首次启动前确认

客户端能使用的前提是服务端已经在服务器电脑上启动。请向管理员确认服务器地址。

示例：

```text
http://192.168.1.20:8080
```

如果服务端改过端口，地址里的端口也要跟着修改。

## 修改服务器地址

如果服务器 IP 变更，在登录页的“审批服务器”区域填写新地址并保存。保存后重新登录。

## 连接自检

登录页和首次连接页可以点击“连接自检”。自检会提示服务器是否可达、服务端版本和地址是否适合局域网使用。

如果看到 `127.0.0.1 只代表当前电脑`，说明当前地址只能在服务器电脑自己使用。请向管理员要服务端窗口中显示的局域网地址，例如：

```text
http://192.168.1.20:8080
```

如果提示版本不兼容，请联系管理员重新分发同一版本的客户端安装包。

## 使用范围

客户端功能与网页工作台一致：

- 设计师提交图纸、配置签名、打印归档。
- 主管和工艺审核图纸。
- 管理员维护目录、用户、日志、备份和签名模板。

数据仍统一保存在审批服务器，不保存在个人电脑客户端里。

## 客户端更新

客户端启动后会自动通过服务端 `/updates/latest.yml` 检查新版，不需要先登录。发现新版时会自动下载客户端安装包并显示进度；下载完成后弹出安装步骤。按提示打开安装包，并根据 Windows 安装向导覆盖安装即可。

客户端不会静默自动安装，也不会向普通用户显示服务端安装包。如果安装向导提示客户端正在运行，请关闭当前客户端窗口后继续安装。

如果当前电脑安装的是 `0.9.0` 或更早版本，需要先由管理员发放并手动安装一次 `0.9.1` 客户端；安装 `0.9.1` 后，后续新版才会在启动时自动检查和下载。

## 打印并归档

设计师在图纸进入“已通过待打印”并生成签后 PDF 后，可在审批详情页点击“打印并归档”。

客户端会先打开打印设置面板，可选择打印机、份数、打印范围、纸张、方向、彩色/黑白、双面、边距、缩放和是否打印背景。点击确认后，客户端调用 Windows 系统打印；系统打印回调成功后，审批记录会自动标记为已打印归档。

注意：客户端能确认打印任务已提交给 Windows，但无法判断打印机后续是否缺纸、卡纸或实际出纸。如果打印取消或失败，系统不会自动归档。浏览器访问时不能调用原生打印参数，请打开签后 PDF 打印后手动标记归档。

## V6.1 图纸批注

主管和工艺审核时，可在审批详情页直接对 PDF 做图纸批注。

- 先选择定位、箭头、矩形、圆形、文字、画笔或云线工具。
- 在 PDF 上标记位置后填写批注内容。
- 选中已有批注后，可移动、缩放和调整颜色。
- 右侧批注列表点击任一条，会自动滚动到图纸对应位置。
- 审查版 PDF 会包含批注，便于归档问题。

设计师登录后可查看批注并标记处理完成，但不能新增审核批注。正式签后 PDF 不包含这些审核批注，仍用于打印签字归档。

## 常见问题

- 无法连接服务器：检查服务器是否开机、审批服务是否运行、防火墙是否放行服务端端口。
- 登录失败：确认账号密码是否正确，或先保存正确服务器地址。
- PDF 打不开：请让管理员检查服务器上的审批文件、签名状态和服务日志。
